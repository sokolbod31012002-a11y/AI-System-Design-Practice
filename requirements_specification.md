# Requirements Specification — «Є-Психолог»

## 1) Project context
**Problem:** В Україні спостерігається критичне зростання попиту на психологічну допомогу. Це призводить до довгих черг та відсутності своєчасного скринінгу для людей у кризових станах.
**System goal:** Створити платформу, де AI-вузол виконує роль "першого контакту" (скринінг та підтримка), виявляючи пацієнтів із високим ризиком (High-Risk).
**In-scope:**
- AI-чатбот для первинного збору анамнезу.
- Предиктивний підбір психолога.
- Система ескалації (Human-in-the-loop) для кризових ситуацій.
**Out-of-scope:** Постановка медичних діагнозів та призначення ліків.

## 3) Functional requirements (FR)
- **FR-01 (AI Скринінг):** AI ставить уточнюючі питання і формує Summary. (Owner: Психолог, Priority: Must)
- **FR-02 (Ескалація):** При виявленні ризику суїциду чат передається людині. (Owner: Супервізор, Priority: Must)
- **FR-03 (Анонімізація):** Видалення імен/телефонів перед відправкою до LLM. (Owner: Кібербезпека, Priority: Must)
- **FR-04 (Метчинг):** Рекомендація 3-х психологів на основі аналізу запиту. (Owner: Користувач, Priority: Should)
- **FR-05 (Логування):** Збереження всіх відповідей AI для аудиту. (Owner: Юрист, Priority: Must)
- **FR-06 (Доступ):** Адмін не бачить змісту приватних чатів. (Owner: Кібербезпека, Priority: Must)

## 4) AI-specific requirements (ASR)
- **ASR-01 (Recall):** Target ≥ 0.98 для High-Risk випадків. Fallback: Keyword-фільтрація.
- **ASR-02 (Latency):** Time to First Token ≤ 1.2 сек. Fallback: Квантована модель.
- **ASR-03 (Explainability):** Підсвічування цитат користувача, що вплинули на Risk Score.
- **ASR-04 (Autonomy):** AI заборонено давати назви ліків. Fallback: Хардкод-заглушка.
- **ASR-05 (Robustness):** Робота з суржиком (відхилення Recall ≤ 5%).
- **ASR-06 (Drift Detection):** Моніторинг зсуву тем раз на 7 днів.

## 5) Assumptions & Constraints
- Дані захищені згідно з HIPAA/GDPR.
- Вартість сесії AI < $0.01.
- Психологи проходять ручну верифікацію дипломів.

## 6) Risks & Mitigations
- Галюцинації AI → Mitigation: Суворий System Prompt + Guardrails.
- Витік даних через промпт → Mitigation: PII-сканер перед відправкою.
- Jailbreak атаки → Mitigation: Окремий шар LLM-Guard.
